# image_format_converter

## Description

- 画像ファイルを他の画像形式に変換するツールです。

## Build

- Makefileからビルドを実行してください。

## Usage

```shell
$ image_format_converter [変換元画像ファイル] [変換後画像ファイル]
```

### Example

```shell
$ image_format_converter sample_image/A.bmp output_image/A.png
```

## Convertible image formats

- bmp
- png
